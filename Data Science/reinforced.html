Robot Vacuum Cleaner Simulation

The AI must learn to clean a small mess (🍂) and a big mess (🗑️) while avoiding furniture (🛋️).

The Learning Loop

1. Observe State 🧐

The agent checks its current position on the grid.

2. Choose Action 💥

It picks a move (up, down, left, right) using a selected strategy to balance exploration and exploitation.

3. Get Reward 🏆

It gets a small reward for (🍂), a large one for (🗑️), a penalty for hitting (🛋️), and a cost for each step.

4. Update Brain 💡

The agent updates its memory (e.g., the Q-table) to remember which actions lead to good or bad rewards in each state.

How Epsilon-Greedy Works:

A simple and effective method. Most of the time, it exploits the best-known action. However, with a small probability (epsilon), it explores by choosing a completely random action to discover new paths.

How UCB Works:

An optimistic approach. It picks actions based not only on their past success but also adds a "bonus" for actions that haven't been tried often. This encourages exploring uncertain but potentially very rewarding options.

How Thompson Sampling Works:

A probabilistic method. For each action, it maintains a probability distribution of the likely reward. To choose an action, it samples a random value from each distribution and picks the action with the highest sample, naturally balancing exploration and exploitation.
