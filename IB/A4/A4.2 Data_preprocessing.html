1. The Impact of Data Quality

Garbage In, Garbage Out (GIGO)

The performance of any machine learning model is fundamentally bottlenecked by the quality of the data it is trained on. Data cleaning is the process of fixing or removing incorrect, corrupted, incorrectly formatted, duplicate, or incomplete data within a dataset.

  • Improves model accuracy and efficiency.
  • Prevents skewed or biased predictions.
  • Reduces computational overhead.

Handling Outliers

Outliers are data points that differ significantly from other observations. They heavily skew models (like linear regression or K-means).

  • Deletion: Remove extreme values entirely.
  • Capping (Winsorizing): Limit extreme values to a specific percentile.

3. Feature Scaling (Normalization & Standardization)

Machine learning algorithms perform better when numerical input variables are scaled to a standard range. Without scaling, features with larger numeric ranges (e.g., Salary vs. Age) will artificially dominate the model.

Normalization (Min-Max Scaling)

Rescales values to a fixed range, usually 0 to 1. Good when the distribution is unknown or not Gaussian.

X' = (X - X_min) / (X_max - X_min)

Standardization (Z-Score)

Centers the data around a mean of 0 with a standard deviation of 1. Better when data contains outliers.

Z = (X - μ) / σ
