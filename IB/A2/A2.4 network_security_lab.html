Network Security

A2.4.1 Firewalls & NAT

Firewalls inspect incoming and outgoing traffic based on predetermined rules. They act as a barrier between a trusted network (LAN) and an untrusted one (Internet).

Rule Types

Whitelist: Default DENY. Only specifically approved traffic is allowed.

Blacklist: Default ALLOW. Specific known threats are blocked.

NAT as Security

Network Address Translation hides internal Private IP addresses from the public Internet, meaning outside attackers cannot directly address internal devices.
