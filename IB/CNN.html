1. The Input Layer

Computers don't see colors or shapes; they see numbers. Every image is converted into a matrix of numerical values representing pixel intensities. This is the starting point for the CNN.

Applications

  • • Digitizing medical X-rays for analysis.
  • • Pre-processing satellite imagery for weather prediction.

2. Convolution (The Filters)

This is the heart of the CNN. Small matrices called Kernels slide across the image using a specific step size called a Stride. At each step, it mathematically detects geometric features, producing a new grid called a Feature Map.

Applications

  • • Facial Recognition: Identifying eyes, nose, and jawlines.
  • • Self-Driving Cars: Detecting road lanes and obstacles.

3. Activation Functions (ReLU)

Real-world data is complex and "non-linear." The ReLU (Rectified Linear Unit) activation function introduces non-linearity by replacing all negative pixel values with zero, clamping useless signals and keeping only the strong positive features.

Applications

  • • Noise Filtering: Removing irrelevant background data.
  • • Deepening Networks: Preventing mathematical degradation.

4. Pooling (Downsampling)

Max Pooling reduces the resolution of the feature map while keeping the most important information. It looks at a small window and keeps only the highest value, making the network faster and less sensitive to small shifts.

Applications

  • • Mobile AI: Compressing models to run on phones.
  • • Translation Invariance: Recognizing a cat even if it moves.

Integration & Classification

Once 2D spatial feature maps are extracted, they undergo Flattening—unrolling the 2D grid into a single 1D vector. This vector is fed into a Dense Layer (Fully Connected Network).

The final layer uses a Softmax function, which squashes raw output scores into a probability distribution (e.g., 85% House, 15% Skyline) that sums to exactly 1.0, giving us our final classification.

Activation Selection

Why do we use ReLU in hidden layers instead of older functions like Sigmoid?

Sigmoid compresses values between 0 and 1. In deep networks, this causes the Vanishing Gradient Problem—where mathematical learning signals shrink so small that the network stops learning. ReLU avoids this by keeping strong gradients for positive values, allowing faster training.

Loss Functions & Deep Learning

A CNN starts out completely "blind" with random kernel weights. How does it learn to find edges or faces automatically? By calculating its mistakes using a Loss Function.

Cross-Entropy Loss

The standard for classification tasks. It heavily penalizes the network when it makes a confident mistake (e.g., predicting 99% Skyline when the image is a Forest), mathematically pushing the probability outputs toward the correct label.

Backpropagation

Once the Loss is calculated, the network works backwards using Calculus. It determines how much every single kernel weight contributed to the error, and slightly adjusts those weights to minimize the loss score on the next pass.

CNN Core Vocabulary

CNN

A specialized neural network designed to process structured grid data, like image pixels.

Input Layer

The raw data fed into the network, representing image pixels as numerical matrices.

Kernel / Filter

A small moving matrix (e.g., 3x3) of weights that detects specific features like edges or textures.

Feature Map

The output grid created after a kernel scans an image, highlighting where patterns were detected.

Spatial Hierarchies

The concept where deeper layers in a CNN combine simple features (edges) from earlier layers into complex objects (faces).

Stride

The number of pixels a kernel shifts as it slides across the image during convolution.

ReLU Activation

A mathematical gate that replaces all negative pixel values with zero to introduce non-linearity.

Max Pooling

A downsampling layer that shrinks the feature map by keeping only the highest value in local patches.

Flattening

Unrolling a 2D feature map into a single 1D list of numbers for the fully connected network.

Dense Layer

A fully connected layer that takes extracted features and learns to make a final classification.

Softmax

A final mathematical function that converts raw scores into probability percentages that sum to 1.0.

Loss Function

A mathematical tool (like Cross-Entropy) that calculates the error or penalty of the network's prediction.

Backpropagation

The algorithmic process of working backward through the network to update weights based on the loss score.
