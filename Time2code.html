Interactive Python 101 Summary

Test your knowledge of Python fundamentals.

1. Variables & Data Types

Variables are names that store data. Python automatically detects the data type. The most common types are strings (text), integers (whole numbers), and floats (decimal numbers).

# A string stores text
course_name = "Python Level 1"

# An integer stores a whole number
participants = 25

# A float stores a number with a decimal
course_rating = 4.7

Your Turn!

If you create a variable `score = 100`, what data type is it?

2. Basic Operations

You can perform arithmetic with numbers and combine (concatenate) strings. Remember Python follows the order of operations (PEMDAS/BODMAS).

# Arithmetic
total_cost = 10 + 5 * 2  # result is 20, not 30

# String concatenation
greeting = "Hello" + " " + "World"

Your Turn!

What is the value of `result = (4 + 6) / 2`?

3. Lists

A list is an ordered collection of items, created with square brackets `[]`. You can access items using their index, which starts at `0`.

topics = ["Variables", "Operations", "Lists"]

# Get the first item (index 0)
first_topic = topics[0]

# Add an item to the end
topics.append("Loops")

Your Turn!

From the initial `topics` list, what index would you use to get the string `"Lists"`?

4. Conditional Statements

`if`, `elif` (else if), and `else` statements allow your code to make decisions and run different blocks of code based on conditions.

grade = 85

if grade >= 90:
    print("Excellent!")
elif grade >= 80:
    print("Good job!")
else:
    print("Keep trying!")

Your Turn!

Based on the code above, what message will be printed?

5. Loops

Loops repeat a block of code. A `for` loop is great for iterating over a sequence, like a list or a range of numbers.

# Loop through a list
for topic in ["Variables", "Lists", "Loops"]:
    print(topic)

# Loop a specific number of times
for i in range(3):  # Will run for i = 0, 1, 2
    print("Hello!")

Your Turn!

How many times will "Hello!" be printed by the `range(3)` loop?

6. Functions

Functions are reusable blocks of code defined with `def`. They can take inputs (parameters) and produce an output (return value).

# Define a function
def add_numbers(a, b):
    return a + b

# Call the function and store the result
sum_result = add_numbers(5, 10) # sum_result is now 15

Your Turn!

What would be the result of calling `add_numbers(7, 8)`?
