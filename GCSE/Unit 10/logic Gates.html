Expression Help

How to write logic expressions.

Supported Operators:

  • AND: &, &&, and, *
  • OR: |, ||, or, +
  • NOT: !, not, ~
  • XOR: ^, xor
  • NAND: nand
  • NOR: nor
  • XNOR: xnor

Variables:

Use single uppercase letters (A, B, C...).

Grouping:

Use parentheses ( ) to group expressions and define order of operations.

Examples:

  • A AND B
  • (A & B) | C
  • !(A ^ B)

How to Use Circuit Builder:

  1. Drag gates from the toolbar onto the grid.
  2. Click a gate to select it, then press `Delete` or `Backspace` to remove it.
  3. Click an output node (right side), then an input node (left side) to connect them.
  4. Click a wire to delete it.
  5. Use 'INPUT' components to control signals. Click their body to toggle state.
  6. Use 'OUTPUT' component (LED) to see final results.
  7. Pan: Hold Middle Mouse Button and drag.
  8. Zoom: Use the Mouse Wheel.
